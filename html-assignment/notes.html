HTML Notes

Headings
We use this tag for the most important text on the page. h1 is the most important and largest heading, with h2 and onward descending in size and importance.
Blockquote
A quote derived from a source, which has the special feature of being able to add the source inside the blockquote.
Emphasis
Used to emphasize text and mark it as somewhat important. Screen readers have a special voice for text with this tag.
Important Text
Used to emphasize text and mark it as highly important. Screen readers have a special voice for text with this tag.
Address
Tags text as part of an address, contact info, phone number, etc.
